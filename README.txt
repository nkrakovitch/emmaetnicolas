SITE DU MARIAGE D’EMMA ET NICOLAS

1. Ouvrez index.html dans un navigateur pour prévisualiser le site.
2. Pour le publier gratuitement, déposez le dossier sur Netlify, GitHub Pages ou Cloudflare Pages.
3. Le formulaire RSVP pointe actuellement vers le Google Form figurant dans vos échanges.
4. Les adresses Google Maps peuvent être ajustées si nécessaire.
5. Pour changer un texte, modifiez index.html. Pour les couleurs et la mise en page, modifiez style.css.
